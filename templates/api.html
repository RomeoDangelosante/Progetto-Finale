Endpoint principali

GET · Recupero dati di mercato

Il bot richiede le candele OHLC dell'oro in tempo reale:

# Richiesta HTTP GET GET /api/v1/rates/XAUUSD?timeframe=H1 # Headers di autenticazione Authorization: Bearer {API_KEY} Content-Type: application/json # Risposta JSON del server { "symbol": "XAUUSD", "timeframe": "H1", "data": [ { "time": "2024-01-15T10:00:00", "open": 2024.50, "high": 2031.80, "low": 2018.30, "close": 2028.90 } ] }

POST · Apertura ordine

Quando il bot rileva un segnale, invia un ordine al broker:

# Richiesta HTTP POST POST /api/v1/orders # Body della richiesta (JSON) { "symbol": "XAUUSD", "action": "buy", "volume": 0.01, "sl": 2010.00, # stop loss "tp": 2060.00 # take profit } # Risposta in caso di successo { "status": "ok", "order_id": 394821, "price": 2028.90, "timestamp": "2024-01-15T10:02:31" }

Chiamata API in Python

Implementazione con libreria requests

import requests import json API_KEY = "il_tuo_api_key" BASE_URL = "https://api.broker.com/v1" def get_candele(simbolo, timeframe="H1", n=100): """Recupera le ultime n candele del simbolo""" url = f"{BASE_URL}/rates/{simbolo}" headers = {"Authorization": f"Bearer {API_KEY}"} params = {"timeframe": timeframe, "count": n} risposta = requests.get(url, headers=headers, params=params) if risposta.status_code == 200: return risposta.json()["data"] else: raise Exception(f"Errore {risposta.status_code}") def apri_ordine(simbolo, azione, volume, sl, tp): """Invia un ordine al broker""" url = f"{BASE_URL}/orders" body = { "symbol": simbolo, "action": azione, "volume": volume, "sl": sl, "tp": tp } risposta = requests.post( url, headers={"Authorization": f"Bearer {API_KEY}"}, json=body ) return risposta.json()
